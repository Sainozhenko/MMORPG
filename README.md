An authentic and modern web portal for an MMORPG (Lineage 2 style) server. This project features a dynamic news system, server status tracking, and a fully functional user dashboard.

---

## 🔥 Key Features

* **⚡ Dynamic News System:** Easily manage game updates through the Django Admin panel.
* **📊 Server Status:** Real-time monitoring of game server populations (Interlude, Multicraft, etc.).
* **👤 User Dashboard:** Personalized profiles for players to manage their accounts.
* **🌍 Multilingual Support:** Ready for international players (English, Ukrainian, Russian).
* **🎨 Responsive Design:** Fully animated and mobile-friendly UI inspired by classic L2 aesthetics.

---

## 🛠️ Tech Stack

| Technology | Usage |
| :--- | :--- |
| **Django 4.2** | Backend Framework |
| **WhiteNoise** | Static Files Management (CSS/JS/Images) |
| **Gunicorn** | Production Web Server |
| **Pandas/Openpyxl** | Data processing & Leaderboards |
| **HTML5/CSS3** | Modern frontend with custom animations |

## 🚀 Quick Start

To run this project locally:

1. **Clone the repository:**
   git clone [https://github.com/Sainozhenko/MMORPG.git](https://github.com/Sainozhenko/MMORPG.git)
   
2. **Install dependencies:

pip install -r requirements.txt

3. **Run migrations:

python manage.py migrate

4. **Start the server:

python manage.py runserver
